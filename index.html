Syntax

/* style.scss or style.less */
h1 {
  color: #0982C1;
}

/* style.sass (ohne klammern)*/
h1
  color: #0982c1
  
/* style.styl */
h1 {
  color: #0982C1;
}

OMG!

Keine Interpunktion!

Syntax Stylus

/* style.styl */
h1 {
  color: #0982C1;
}
/* ohne Klammern */
h1
  color: #0982C1;
  
/* Ohne  colons and semi-colons */
h1
  color #0982C1
